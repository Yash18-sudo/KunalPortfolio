
# Personal portfolio
This is my personal portfolio website where on can know more about my work and achievements.

## Tech Stack
- React
- Styled components

## How to use
1. Clone the repo
``` bash
git clone https://github.com/Yash18-sudo/Portfolio.git
```

2. Install dependencies
``` bash
npm install
```

3. Start the dev server
``` bash
npm start
```
